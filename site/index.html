包含 4 个工程化 Skill

每个 skill 解决一类完整问题，含真实项目踩坑经验

scaffolding

pi-template-create

从零创建一个新模板类型。目录脚手架 → 端口规划 → 双模式配置 → CLI 注册（5 处必改）→ 同步脚本 → 测试。含 browser-agent 半接入陷阱的真实教训。

触发词：新建模板add template照着做一个
backend

pi-rpc-module-dev

给模板加新功能的完整 5 步流程：类型定义 → Handler → 注册 → 前端调用 → 测试。含 6 条 ESLint 强制规则和常见错误避坑。

触发词：新增功能add module新 RPC 方法
frontend

pi-template-dev

模板内部前端开发规范。30 个 CSS 变量主题系统、i18n 国际化、Zustand Store、组件开发、Commit 规范。

触发词：组件样式主题i18n
debugging

pi-fullstack-debug

全栈调试指南。React → Store → RPC → Transport → Handler 全链路排查。6 大问题分类 + Token/安全拦截调试方法。

触发词：白屏断连不工作报错
