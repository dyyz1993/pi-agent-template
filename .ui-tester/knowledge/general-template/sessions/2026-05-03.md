## Findings
- Page has critical stability issue: crashes ~4-5s after load
  - Root cause: form submission on Enter key navigates page to about:blank
  - Fix: prevent form submit + beforeunload events
  - localStorage debug-panel-collapsed=true also contributed to crashes on reload
- ActivityBar uses title= attribute (not aria-label) unlike the agent-template on port 5175
- File tree items not in accessibility tree — must use find text or eval
- No Clear button in Debug panel for General template (present in agent-template)
